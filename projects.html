Yoster Island: repositório no Github

Jogo educacional desenvolvido em C através da biblioteca Allegro. Projeto Integrador do curso de Ciência da Computação do Centro Universitário Senac.

Yoster Island foi um projeto feito no segundo semestre do curso com o intuito de ser um jogo educacional baseado na Teoria da Evolução, de Charles Darwin. É um jogo offline desenvolvido em um sistema Linux. No processo de desenvolvimento, foi utilizado um script em shell para "acelerar" o processo de compilação.

Kurisu-Varsóvia: repositório no Github

Sistema autônomo desenvolvido em C#. Projeto Integrador do curso de Ciência da Computação do Centro Universitário Senac.

O projeto em si se baseia em uma estratégia para o Cartagena, um jogo de tabuleiro, onde foi desenvolvido um conjunto de regras. Esse sistema autônomo se divide em alguns estágios: Emular as jogadas possíveis, distribuir pontuações para cada uma (sendo maior melhor), armazená-las em uma Fila de Prioridades (estrutura de dados) ordenada e retirar a com maior pontuação dessa fila para, então, ser jogada.
